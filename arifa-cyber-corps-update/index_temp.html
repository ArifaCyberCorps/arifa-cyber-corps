Projects

How We Build Real Cybersecurity Projects

From research to real-world implementation, our projects focus on solving practical cybersecurity problems through hands-on development.

View on GitHub

SIEM AI Assistant

An intelligent assistant designed to support SIEM operations by analyzing security logs, highlighting anomalies, and assisting with faster incident understanding.

TOR-Unveil

A research-driven project focused on analyzing TOR network behavior by correlating relay data and traffic patterns to support cybercrime investigation and forensic analysis.

Dynamic Security Policy Enforcer

A security-focused project that dynamically applies and updates security policies based on system behavior, detected risks, and predefined rules to reduce attack exposure.
